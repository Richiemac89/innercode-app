ICON REQUIREMENTS FOR PWA/APP STORE

For a production-ready mobile app, you'll need the following icons:

1. icon-192.png (192x192px) - Android home screen
2. icon-512.png (512x512px) - Android splash screen
3. apple-touch-icon.png (180x180px) - iOS home screen
4. favicon.ico (32x32px) - Browser tab icon
5. og-image.png (1200x630px) - Social media preview

INSTRUCTIONS:
- Create your logo/icon design
- Use a tool like https://realfavicongenerator.net/ to generate all sizes
- Or use Figma/Photoshop to export each size manually
- Replace the placeholder icons in this folder

DESIGN TIPS FOR MOBILE ICONS:
- Use simple, bold designs (avoid fine details)
- Ensure it looks good at small sizes
- Use high contrast
- Avoid text if possible (it becomes unreadable)
- Test on actual devices
